# GoMob

GoMob adalah projek pengembangan sistem absensi berbasis lokasi yang dibuat menggunakan React Native.
Aplikasi ini dirancang untuk memudahkan proses pencatatan kehadiran pegawai dengan memanfaatkan fitur perangkat mobile seperti GPS dan kamera.

> [!IMPORTANT]
> 🚧 Projek ini masih berada dalam tahap pengembangan aktif dan terus dikembangkan secara bertahap.

## <img src="https://media3.giphy.com/media/v1.Y2lkPTZjMDliOTUyeHU1MG5jMHMzZDhpNjdmdTl6OWppeHY1d3R3NHljcnR6ZGttczRsNCZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/12FBrUdUj7ZkuzyI8G/giphy.gif" width="35"><b> Tujuan Projek </b>

Proyek ini dimulai dengan selintas ide untuk membuat sistem absensi pegawai internal, yang dikhusus-kan untuk para pegawai Non ASN. GoMob dibuat untuk mengembangkan sistem absensi yang :

- Praktis digunakan melalui perangkat mobile
- Memiliki validasi lokasi berbasis GPS
- Mengurangi potensi manipulasi absensi
- Mempermudah monitoring kehadiran pegawai

Beberapa fitur yang sedang dan telah dikembangkan:

- Login pengguna
- Navigasi aplikasi (Dashboard, Absen, Profile)
- Absensi berbasis lokasi GPS
- Validasi radius lokasi kerja
- Perhitungan jarak pengguna dengan lokasi kerja
- Integrasi tampilan peta
- Pencatatan waktu absensi
